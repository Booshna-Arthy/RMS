Create New Class

Start here to create a new class. Classes are placed inside courses.

You can place X classes in a course.

You can work on the class and then add it to the single or multiple courses.

Create New Quiz

Start here to create a new quiz. It is to test a user's understanding on the course that was assigned to him.

Based on the scoring, system will determine the pass or fall percentage.

Create New Course

Start here to create a new course. You can then fill it with classes.

Upon course completion a certificate will be awarded.

Complete progress of each class is tracked and can br seen in the Reports menu

Create New Notification

Start here to create a new notification. You can notify employees on important announcements and also receive their acknowledgement on the same.
